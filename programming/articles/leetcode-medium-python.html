1. Overview

This article summarizes common solutions for medium-level problems by reusable patterns. Each example is clean, iterative Python suitable for interviews.

Pattern → Problem Index (Try These First)

  • Sliding Window: strings/arrays with “longest/shortest/at most K/without repeating” (e.g., longest substring without repeat, at most K distinct, min window).
  • Two Pointers: sorted arrays, pair/sum/closest problems, partitioning; choose when inputs are sorted or can be sorted.
  • Merge Intervals: calendar/meeting rooms, insert/merge intervals, summary ranges, non-overlapping scheduling.
  • Monotonic Stack: next greater element, daily temperatures, stock span, largest rectangle in histogram.
  • Prefix Sum + Hash: subarray sum equals K, equal 0s/1s, divisible by K, range counts; when subarray sums are involved.
  • Heap / Top-K: k most frequent, kth largest, merge k sorted lists/arrays, streaming medians.
  • Binary Search (value or answer): search in rotated array, first bad version, capacity to ship, min speed to eat bananas; when function is monotonic.
  • BFS: shortest path on unweighted grid/graph, level-order traversals, islands with step counts.
  • Union-Find: number of islands (static), accounts merge, redundant connection, connectivity queries.
  • Topological Sort: course schedule, build/package ordering, prerequisite dependency resolution in DAGs.

2. Order of Code (Template Mindset)

  • Binary Search: init lo/hi → while lo ≤ hi → mid → pick side → maintain invariant.
  • Two Pointers: sort/init → l/r → move one pointer per rule → update answer.
  • Sliding Window: expand right → while invalid shrink left → update best.
  • Monotonic Stack: iterate → while violates pop and compute → push.
  • Prefix Sum / Hash: maintain running sum/map → query/update per step.

3. Binary Search (Leftmost/Rightmost)

def lower_bound(a, x):  # first idx with a[i] >= x
    lo, hi = 0, len(a)-1
    ans = len(a)
    while lo <= hi:
        mid = (lo+hi)//2
        if a[mid] >= x:
            ans = mid; hi = mid-1
        else:
            lo = mid+1
    return ans

def upper_bound(a, x):  # first idx with a[i] > x
    lo, hi = 0, len(a)-1
    ans = len(a)
    while lo <= hi:
        mid = (lo+hi)//2
        if a[mid] > x:
            ans = mid; hi = mid-1
        else:
            lo = mid+1
    return ans

4. Two Pointers

def two_sum_sorted(a, target):
    i, j = 0, len(a)-1
    while i < j:
        s = a[i] + a[j]
        if s == target: return [i, j]
        if s < target: i += 1
        else: j -= 1
    return []

2. Stack Pattern

Use a stack to match opening/closing delimiters or to keep candidates that may be invalidated later.

def valid_parentheses(s: str) -> bool:
    pairs = {')': '(', ']': '[', '}': '{'}
    st = []
    for ch in s:
        if ch in pairs:
            if not st or st.pop() != pairs[ch]:
                return False
        else:
            st.append(ch)
    return not st

5. Sliding Window

Grow/shrink a window while maintaining an O(1) check that determines whether to advance left.

def longest_substring_no_repeat(s: str) -> int:
    seen = set(); left = 0; best = 0
    for right, ch in enumerate(s):
        while ch in seen:
            seen.remove(s[left]); left += 1
        seen.add(ch)
        best = max(best, right - left + 1)
    return best

6. Merge Intervals

def merge_intervals(iv):
    iv.sort()
    out = []
    for s, e in iv:
        if not out or s > out[-1][1]:
            out.append([s, e])
        else:
            out[-1][1] = max(out[-1][1], e)
    return out

7. Monotonic Stack

def next_greater(nums):
    res = [-1]*len(nums)
    st = []  # indices, decreasing stack by value
    for i, x in enumerate(nums):
        while st and nums[st[-1]] < x:
            res[st.pop()] = x
        st.append(i)
    return res

8. Prefix Sum

Prefix sums turn many range questions into simple lookups. A classic interview task counts subarrays whose sum equals k using a hash map of prefix frequencies.

# count subarrays with sum == k
def subarray_sum(nums, k):
    from collections import defaultdict
    cnt = defaultdict(int); cnt[0] = 1
    run = ans = 0
    for x in nums:
        run += x
        ans += cnt[run - k]
        cnt[run] += 1
    return ans

Capacity Burning Difference Array

Range-update problems such as Car Pooling benefit from the difference-array flavour of prefix sums. We mark passenger deltas at each stop and "burn" remaining capacity as we scan the timeline—dropping to a negative value means we over-booked the car.

from typing import List

def can_carpool(trips: List[List[int]], capacity: int) -> bool:
    diff = [0] * 1001  # net passenger change at each stop
    for passengers, start, end in trips:
        diff[start] += passengers
        diff[end] -= passengers

    remaining = capacity
    for delta in diff:
        remaining -= delta  # subtract pickups, add drop-offs
        if remaining < 0:
            return False
    return True

9. Heap / Top-K

Use heapq.nlargest with a key function based on a Counter for top-k frequent elements.

from collections import Counter
import heapq

def top_k_frequent(nums, k):
    freq = Counter(nums)
    return heapq.nlargest(k, freq.keys(), key=freq.get)

10. Union-Find

class DSU:
    def __init__(self, n):
        self.p = list(range(n)); self.r = [0]*n
    def find(self, x):
        while self.p[x] != x:
            self.p[x] = self.p[self.p[x]]
            x = self.p[x]
        return x
    def union(self, a, b):
        ra, rb = self.find(a), self.find(b)
        if ra == rb: return False
        if self.r[ra] < self.r[rb]: ra, rb = rb, ra
        self.p[rb] = ra
        if self.r[ra] == self.r[rb]: self.r[ra] += 1
        return True

11. BFS Level Order

Level-order traversal uses a queue and processes each level’s breadth before moving on.

from collections import deque

def level_order(root):
    if not root: return []
    out, q = [], deque([root])
    while q:
        n = len(q); level = []
        for _ in range(n):
            node = q.popleft()
            level.append(node.val)
            if node.left: q.append(node.left)
            if node.right: q.append(node.right)
        out.append(level)
    return out

12. Topological Sort (DAG)

Detect whether all courses can be completed by Kahn’s algorithm (BFS on in-degrees).

from collections import defaultdict, deque

def can_finish(num_courses, prerequisites):
    g = defaultdict(list)
    indeg = [0]*num_courses
    for a, b in prerequisites:
        g[b].append(a); indeg[a] += 1
    q = deque([i for i in range(num_courses) if indeg[i]==0])
    done = 0
    while q:
        u = q.popleft(); done += 1
        for v in g[u]:
            indeg[v] -= 1
            if indeg[v]==0: q.append(v)
    return done == num_courses

13. Array Tricks

  • Counts: subarrays = n*(n+1)//2; pairs nC2 = n*(n-1)//2; subsets = 2**n.
  • Between bounds: exclusive count R-L-1; inclusive R-L+1.
  • Circular array: next = (i+1)%n; prev = (i-1+n)%n; circular prefix via doubling.
  • Content-as-index: use sign marking on 1..n arrays for duplicates/missing.
# Circular prefix sum by doubling
def circular_prefix(nums):
    n = len(nums)
    ps = [0]*(2*n+1)
    for i in range(2*n):
        ps[i+1] = ps[i] + nums[i % n]
    return ps

# Boyer–Moore Majority Vote
def majority_element(nums):
    count = 0; cand = None
    for x in nums:
        if count == 0: cand = x
        count += 1 if x == cand else -1
    return cand

# Rotate array k steps using 3 reversals (in-place)
def rotate(nums, k):
    k %= len(nums)
    def rev(i, j):
        while i < j:
            nums[i], nums[j] = nums[j], nums[i]; i += 1; j -= 1
    rev(0, len(nums)-1); rev(0, k-1); rev(k, len(nums)-1)

# Find duplicates using sign marking (1..n domain)
def find_duplicates(nums):
    res = []
    for x in nums:
        i = abs(x) - 1
        if nums[i] < 0: res.append(abs(x))
        nums[i] = -nums[i]
    return res

# Product of array except self (O(1) extra)
def product_except_self(nums):
    n = len(nums); ans = [1]*n
    for i in range(1, n): ans[i] = ans[i-1]*nums[i-1]
    r = 1
    for i in range(n-1, -1, -1): ans[i] *= r; r *= nums[i]
    return ans

# Summary ranges (merge consecutive)
def summary_ranges(nums):
    res = []; i = 0
    while i < len(nums):
        j = i
        while j+1 < len(nums) and nums[j+1] == nums[j]+1: j += 1
        res.append(str(nums[i]) if i==j else f"{nums[i]}->{nums[j]}")
        i = j+1
    return res

14. Daily Divine (Habits & Naming)

  • Big-O practice: quiz regularly; know typical costs per pattern.
  • Names as DSL: prefer lo/hi, left/right, rank/parent over i/j/k.
  • Odd partitions: when splitting odds/evens, use ceil with (n+1)//2.
  • Loop wisdom: the last line of the loop influences the next iteration condition—connect them.
  • DSU practice: path compression + union by rank; keep a minimal DSU handy.

15. Common Mistakes

  • Off-by-one on slices: deleting last 4 chars in Python: s = s[:-4].
  • Inclusive vs exclusive: window length is R-L+1; don’t use R-L.
  • Binary search exits: use tested lower/upper-bound templates; don’t hand-roll variants mid-interview.
  • Rotate array: verify k %= n and segment boundaries.

16. Combinatorics & Off-by-One

  • Inclusive vs exclusive: window [L,R] length is R-L+1. Update after shrinking.
  • Lower/upper bound: leftmost ≥ x vs first > x. Count of x in sorted = ub(x)-lb(x).
  • Prefix sums: subarray [i..j] = pref[j]-pref[i-1]; define pref[0]=0 to avoid underflow.
  • nCk sanity: pairs nC2 = n*(n-1)//2, triplets nC3 = n*(n-1)*(n-2)//6.
  • Grid bounds: check 0 ≤ r < R and 0 ≤ c < C before access.
Emphasize order: decide invariants first; each iteration maintains them. This reduces off-by-one errors.
