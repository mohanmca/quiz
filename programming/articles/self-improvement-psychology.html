1) Motivation (Research + Quotes)

Motivation is not a single force; it emerges from expectations, values, identity, and context. Three research pillars are especially practical:

  • Self‑Determination Theory (Deci & Ryan): Autonomy, Competence, Relatedness. Design work to satisfy these needs.
  • Expectancy–Value (Wigfield & Eccles): We act when we expect success and value the outcome. Raise clarity and make value vivid.
  • Implementation Intentions (Gollwitzer): “If situation X, then I will do Y.” Pre-decisions beat willpower.
“We are what we repeatedly do. Excellence, then, is not an act but a habit.” — Will Durant (summarizing Aristotle)
“The impediment to action advances action. What stands in the way becomes the way.” — Marcus Aurelius
Quick wins: Write one implementation intention per goal. Make the value concrete: Who benefits? How soon? What metric improves?

2) Psychology → Productivity

  • Attention Residue (Sophie Leroy): Context switches leave cognitive “residue”. Batch similar tasks; protect deep work blocks.
  • Flow (Csikszentmihalyi): Match challenge to skill, clear goals, immediate feedback. Reduce friction and ambiguity.
  • Zeigarnik Effect: Open loops persist in memory. Externalize tasks; use a trusted list; close loops with a clear “done”.
  • Parkinson’s Law: Work expands to fill time. Timebox and shrink scope.
“Clarity about what matters provides clarity about what does not.” — Cal Newport

3) Reducing Procrastination

  • Temporal Discounting: We overvalue now vs. later. Make rewards immediate: a small treat after a work sprint.
  • Mood Repair (Sirois & Pychyl): Procrastination often avoids negative feelings. Start tiny to lower discomfort.
  • 2‑Minute Rule: Begin with 120 seconds. Momentum beats hesitation.
  • Precommitment & Friction: Block distractions; schedule focus sessions with someone else (“body‑double”).
Script your first action: “Open file, write a title, list 3 bullets.”

4) Small Continuous Improvement

  • Kaizen / Marginal Gains: 1% daily compounds. Track one tiny improvement per day.
  • Habit Stacking (James Clear): After current habit X, I will do Y. Piggyback to reduce friction.
  • Feedback Loops: Review weekly; promote what worked; prune what didn’t.
“Success is the product of daily habits — not once‑in‑a‑lifetime transformations.” — James Clear

5) Better at Getting Things Done

  • Definition of Done: Write the acceptance criteria before starting.
  • Eisenhower Matrix: Urgent vs. Important. Protect important/non‑urgent work.
  • Checklists (Atul Gawande): Prevent cognitive slips on repeatable tasks.
  • Timeboxing + Review: Work in sprints; end with a 2‑minute reflection.

6) Behavioral Science to Improve Self

  • Fogg Model (B=MAP): Behavior happens when Motivation, Ability, and Prompt coincide. Lower ability barrier first.
  • Nudges & Defaults: Put fruits on desk; hide distracting apps; auto‑open your task doc at login.
  • Identity‑Based Habits: “I am the kind of person who ships.” Cast one vote daily for that identity.
“Whether you think you can or you think you can’t, you’re right.” — Often attributed to Henry Ford

7) Improving Focus

  • Single‑task Windows: 25–50 minutes, one goal, notifications off.
  • Environment: Visual minimalism; full‑screen; noise control; ready checklist.
  • Attentional Warm‑up: 3 minutes of planning: goal, first step, blockers.
  • Recovery: Short breaks, movement, water; beware attention residue on switching.

8) Building Self‑Confidence

  • Self‑Efficacy (Bandura): Confidence grows from mastery experiences. Start small, escalate challenge.
  • Growth Mindset (Dweck): Treat mistakes as data. Use “not yet.”
  • Self‑Compassion (Neff): Speak to yourself as to a friend; reduces rumination, improves persistence.
“Do not wish for an easy life; wish for the strength to endure a difficult one.” — Bruce Lee

9) Increasing Active Listening

  • Rogers’ Core Conditions: Empathy, congruence, positive regard. Aim to understand before responding.
  • Reflect & Label: Paraphrase what you heard, then ask, “Did I get that right?”
  • Nonverbal: Eye contact, nods, short verbal encouragers. Minimal interruptions.
  • Summarize & Next: Close with agreed actions; confirm ownership and timelines.
“Most people do not listen with the intent to understand; they listen with the intent to reply.” — Stephen R. Covey

10) Python Tools for Habits

Quick, practical Python helpers you can run locally to support habits and focus.

1) Micro‑Commit Journal (2 minutes)

from datetime import datetime

def log_commit(msg: str, path: str = "commit_journal.txt"):
    ts = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    line = f"[{ts}] {msg}\n"
    with open(path, "a", encoding="utf-8") as f:
        f.write(line)
    return line

if __name__ == "__main__":
    # Example usage: record a 2-minute start
    print(log_commit("Started: write first paragraph for report (2-min rule)"))

2) Simple Pomodoro Planner

from dataclasses import dataclass
from datetime import datetime, timedelta

@dataclass
class Block:
    start: datetime
    end: datetime
    label: str

def plan_pomodoros(start: datetime, n=4, work=25, break_min=5):
    plan = []
    t = start
    for i in range(n):
        w_end = t + timedelta(minutes=work)
        plan.append(Block(t, w_end, f"Focus #{i+1}"))
        t = w_end
        if i != n-1:
            b_end = t + timedelta(minutes=break_min)
            plan.append(Block(t, b_end, "Break"))
            t = b_end
    return plan

if __name__ == "__main__":
    now = datetime.now()
    for b in plan_pomodoros(now, n=3, work=30, break_min=5):
        print(b.start.strftime('%H:%M'), '→', b.end.strftime('%H:%M'), '-', b.label)

3) Habit Stacking Prompter

def habit_stack(current: str, new: str) -> str:
    return f"After I {current}, I will {new}."

if __name__ == "__main__":
    print(habit_stack("brew coffee","write 3 bullets for daily goal"))

4) Active Listening Checklist (Console Prompt)

QUESTIONS = [
    "What I heard you say is …",
    "What outcome would be ideal for you?",
    "What would make this a win this week?",
    "Is there anything I’m missing?",
]

def guide():
    print("Active Listening Prompts:\n")
    for i, q in enumerate(QUESTIONS, 1):
        print(f"{i}. {q}")

if __name__ == "__main__":
    guide()

12) Scenarios & Model Answers

Short, practical answers that mirror the scenario-style quiz items.

  • Procrastinated 3‑hour report: Start with a 2‑minute outline and 3 bullets. Momentum lowers avoidance and creates a mastery experience.
  • Meetings all day: Batch meetings; protect 1–2 deep work blocks. Capture next actions before switching to cut attention residue.
  • Daily reading habit: Habit stack: “After I brew coffee, I read 10 pages.” Raise to 20 when stable.
  • Tasks lack closure: Define DoD upfront (e.g., PR merged, tests pass, changelog updated). Close loops weekly.
  • Forced context switch mid‑task: Jot 1–2 next steps, then switch. Reduces cognitive load and speeds re‑entry.
  • Under‑challenged and bored: Nudge challenge up and add clear success criteria to enter flow.
  • Teammate venting: Paraphrase and label feelings: “What I’m hearing is … Did I get that right?”
  • Morning workouts stickiness: Reduce friction: lay out clothes/shoes; place alarm by them; prompt on wake.
  • Stretch project fear: Ship a small prototype this week to build self‑efficacy via mastery.
  • Harsh self‑criticism after a miss: Use self‑compassion + quick “what did I learn?” Then schedule the next tiny step.
  • Need 45‑minute deep focus: Full‑screen, notifications off, one goal, first action scripted.
  • Retro without outcomes: Use a checklist: wins, one kaizen, owners and timelines.
