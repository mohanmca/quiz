2. GC Concepts

Generational hypothesis: most objects die young. Parallel/Concurrent collectors (G1/ZGC/Shenandoah) reduce pause times.

# Python analogy: simulate reachability marking on an object graph
from collections import deque

def reachable(graph, roots):
    seen = set(roots); q = deque(roots)
    while q:
        u = q.popleft()
        for v in graph.get(u,()):
            if v not in seen:
                seen.add(v); q.append(v)
    return seen
