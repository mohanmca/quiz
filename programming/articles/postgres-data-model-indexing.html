1. Normalization

1NF/2NF/3NF minimize redundancy and anomalies; denormalize selectively with materialized views or indexes.

2. Index Types

B-Tree for equality/range on sortable types; GIN for tsvector/jsonb containment; GiST for ranges/KNN; BRIN for large, naturally ordered tables.

3. Partial/Expression Index
