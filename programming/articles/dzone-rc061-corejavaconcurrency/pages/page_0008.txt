CONCURRENT COLLECTIONS
The easiest way to make a collection thread-safe is to use 
Collections#synchronized family methods. Because this 
solution performs poorly under high contention, java.util.
concurrent provides a variety of data structures which are 
optimized for concurrent usage.

It usually acts as a bucketed hash
