ConcurrentLinkedQueue

An unbounded blocking queue

IMPLEMENTATION
DESCRIPTION
DelayQueue
An unbounded blocking queue of 
elements, each with a delay value. 
Elements can only be removed 
when their delay has passed and 
are removed in the order of the 
oldest expired item.
SynchronousQueue
A 0-length queue where the 
producer and consumer block 
until the other arrives. When 
both threads arrive, the value is 
transferred directly from producer 
to consumer. Useful when 
transferring data between threads.
