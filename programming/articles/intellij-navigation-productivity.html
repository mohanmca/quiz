1. Overview

Mastering navigation in IntelliJ IDEA compounds developer speed. Focus on search-first workflows, usage/hierarchy views, and Git context. The examples assume Bazel projects in C++ and Scala with multiple implementations per symbol.

2. Search Everywhere & Navigate

  • Search Everywhere: Shift Shift — search files, classes, symbols, actions. Prefix with / to search actions.
  • Go to Class: macOS Cmd+O
  • Go to File: macOS Cmd+Shift+O
  • Go to Symbol: macOS Cmd+Alt+O
  • Recent Files: macOS Cmd+E
  • Navigate Back/Forward: macOS Cmd+[ / Cmd+]
  • Last Edit Location: macOS Cmd+Shift+Backspace
  • File Structure popup: macOS Cmd+F12

3. Declarations, Implementations, Types

  • Go to Declaration/Definition: macOS Cmd+B
  • Go to Type Declaration: macOS Cmd+Shift+B
  • Go to Implementation(s): macOS Cmd+Alt+B (great when multiple impls exist)
  • Super Method / Super Class: macOS Cmd+U
  • Quick Definition: macOS Cmd+Shift+I
  • Quick Documentation: macOS F1

4. Find/Show Usages & Callers

  • Find Usages: macOS Option+F7
  • Show Usages (popup): macOS Option+Cmd+F7
  • Highlight Usages in File: macOS Cmd+Shift+F7
  • Find in Files: macOS Cmd+Shift+F
  • Replace in Files: macOS Cmd+Shift+R

5. Type/Method/Call Hierarchies

  • Type Hierarchy: macOS Ctrl+H
  • Method Hierarchy: macOS Cmd+Shift+H
  • Call Hierarchy: macOS Ctrl+Alt+H
  • Use the hierarchy tool window filters (implementations only, callers only) to prune noise.

6. Scala: Implicits, Traits, Extensions

  • Show Implicit Hints/Conversions: View → Appearance → Show Implicit Hints; or toggle from the editor gutter.
  • Implicit Parameters popup: hover the hint or use Quick Documentation on the call site.
  • Go to Implementations of Trait: use Cmd+Alt+B on the trait name.
  • Type Info: macOS Cmd+Shift+P (Scala type under cursor).
  • Extensions/Typeclass patterns: prefer Find Usages + Call Hierarchy from the typeclass method to enumerate instances.

7. C++: Headers/Sources, Symbols

  • Switch Header/Source: Navigate → Related Symbol (often Cmd+Alt+Home or via intention Alt+Enter).
  • Symbol Navigation: prefer Go to Symbol for overloaded functions; then Go to Implementations to fan out across targets.
  • Macro/Template instantiations: use Quick Documentation and Show Usages to locate specializations.

8. Git: Annotate/Blame, Diff, Log

  • Annotate/Blame: right‑click gutter → Annotate; shows last commit per line. Use when understanding why code exists; jump to commit with Enter.
  • Blame vs History: Blame for line‑level authorship; Git → Show History for file‑level evolution; Git → Show Diff to compare branches/tags.
  • Use cases: regressions (who changed this line), style drifts (when did pattern change), risky refactors (follow across renames via History).
  • Shortcut: open VCS popup: macOS Ctrl+V (Keymap dependent).

9. Bazel: Build/Run/Sync

  • With the Bazel plugin, use the Bazel tool window to Sync project and discover targets.
  • Run Anything: Cmd Cmd — run bazel targets, tasks, and configs quickly.
  • Context actions: Alt+Enter on BUILD files for quick fixes; use gutter run icons on rules where available.

10. Improve Navigation Ability

  • Daily 10-minute drills: pick a symbol and enumerate: declaration → type → implementations → usages → callers → commit.
  • Reduce mouse: force yourself to reach features via shortcuts for a week; print the cheatsheet.
  • Practice filters: in hierarchies/usages, toggle scope (project/test), library filters, and grouping options.
  • Refine Search Everywhere: use action prefix / (e.g., /Invalidate Caches), and camelHumps in class/file search.

12. Memory Techniques

This section distills a cartoon-style mnemonic system and peg associations to memorize macOS IntelliJ shortcuts quickly and retain them under pressure.

12.1 Core Modifier Keys as Characters

Think of each modifier key as a cartoon character with a personality. When a shortcut combines keys, imagine the characters teaming up.

  • Ctrl (Control) → 🕴️ The Boss. Wants to control everything. If a shortcut has Ctrl, think: “I need the Boss to direct this.”
  • Cmd (Command) → 🫡 The General. Commands an army, icon looks like a fort (⌘). If a shortcut has Cmd, imagine issuing a command.
  • Option (Alt) → 🎭 The Trickster. Gives options or alternate paths. If you see Option, think: “I want an alternate version.”
  • Shift → 🏋️ The Bodybuilder. Makes things bigger/stronger/uppercase. If Shift appears, think: “We’re going bigger or adding power.”

12.2 Funny Peg-Style Associations

Glue IntelliJ actions onto these characters as short scenes:

  • Search Everywhere → Cmd+Shift+A: The General climbs on the Bodybuilder with a megaphone yelling, “Search everywhere!”
  • Go to Class → Cmd+O: The General marches into class saying “O-kay cadets, Go to Class!”
  • Go to File → Cmd+Shift+O: The General carries a huge O-shaped file cabinet on the Bodybuilder’s back.
  • Navigate Back/Forward → Cmd+[ / Cmd+]: The General uses square-bracket “doors” to step back and forward in a hallway.
  • Recent Files → Cmd+E: The General pops an Energy Drink (E) to recall recent work.
  • Declaration → Cmd+B: Bounce a huge B‑ball onto code to jump to its declaration.
  • Quick Fix → Option+Enter: The Trickster slams Enter, laughing “Surprise fix!”
  • Search in Path → Cmd+Shift+F: Flashlight (F) while the Bodybuilder lifts the General to search the whole project.
  • Rename → Shift+F6: The Bodybuilder flexes his six‑pack while announcing the new name.

12.3 How to Drill (Memory Pegging)

  • Assign voices: Ctrl = angry boss; Cmd = military general; Option = sneaky joker; Shift = gym bro.
  • Act scenes out: e.g., Cmd+Shift+F → General + Bodybuilder + Flashlight → Search in Path.
  • Exaggerate: The sillier the scene, the stickier the memory trace.

12.4 Extra Retention Tips

  • Spaced repetition: Make an Anki deck of your most-used 20 actions (front: “Find Usages”; back: Option+F7 + your scene). Review 1–2 minutes daily.
  • Chunk by workflow: Drill sequences, not isolated keys. Example: Declaration → Type → Implementations → Usages → Callers.
  • Micro-drills in context: Before running tests, force using Cmd+Shift+R and Cmd+Shift+F instead of menus.
  • Create visual pegs: Print the macOS cheatsheet and annotate with your scenes (emoji, doodles).
  • Weekly review: Pick 5 new shortcuts; add pegs and enforce them during work for a week.
